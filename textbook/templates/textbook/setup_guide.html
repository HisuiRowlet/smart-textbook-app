🛠️ AI Setup Guide

1. Install Ollama

Download and install Ollama from ollama.ai

2. Pull a Model

Open terminal and run one of these commands:

# For smaller, faster models:
ollama pull llama2:7b
# or
ollama pull gemma:2b

# For better quality (requires more RAM):
ollama pull llama2
# or  
ollama pull mistral

3. Start Ollama

Ollama should start automatically. If not, run:

ollama serve

4. Check Status

Visit the AI Status page to verify everything is working.
